# Bashavara

Bashavara is a System Design LAB project which provide users the following features

* Register
* Log In
* Log Out
* WishList
* Property Alert
* Post Advertisement
* Advanced Search 

Web Interface is developed with **PHP Laravel framework, Bootstrap and MySQL database**.
